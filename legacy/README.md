# Legacy Code

This folder contains pre-`src/` modules that were no longer active import targets after the route and feature cleanup.

Keep these files here for reference while the app stabilizes. Do not add new production imports from `legacy/`; move code into `src/components`, `src/features`, `src/services`, `src/hooks`, `src/providers`, `src/utils`, `src/types`, or `src/constants` first.
